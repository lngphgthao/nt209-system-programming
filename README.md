## Repository Structure

The repository is organized into the following directories:

- `books/`: Contains reference materials and textbooks related to system programming.
- `slides/`: Contains lecture slides and presentation materials used during the course.
- `th/`: Contains lab exercises and assignments related to system programming.
- `README.md`: This file, providing an overview of the repository and its contents.
